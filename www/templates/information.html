MODE KENDALI

Terdapat dua mode kendali dalam sistem ini, yaitu kendali manual dan kendali cerdas. Pada mode kendali manual, pengguna dapat menyalakan dan mematikan kipas/ lampu secara manual dengan menekan button. Pada kendali cerdas, sistem akan menjalankan skenario pengambilan keputusan sehingga dalam kondisi tertentu pengguna tidak perlu menekan button secara manual.

Hardware

Sensor Arus AC/DC ACS712-30A

ACS712 bekerja berdasarkan prinsip Hall Effect. Prinsip kerja dari sensor ini adalah arus hasil pembacaan mengalir melalui kabel tembaga yang kemudian menghasilkan medan magnet kemudian ditangkap oleh Hall IC dan diubah menjadi tegangan proporsional. Ketepatan pada tegangan output diperoleh dari BiCMOS IC hall yang diprogram agar memiliki akurasi yang baik. Sensor ini memiliki pembacaan sensor dengan presisi yang tinggi, karena di dalamnya terdapat rangkaian low-offset liner Hall dengan satu lintasan yang terbuat dari tembaga.

Relai Digital 1-Channel

Relai merupakan aktuator yang bekerja berdasarkan prinsip saklar. Terdapat dua jenis mode operasi relai yaitu NO (Normally Open) dan NC (Normally Close). Pada project ini mode operasi yang digunakan yaitu NC dimana relai akan memutus beban ketika diberi logika LOW dan akan menyambungkan beban ketika diberi logika HIGH.

Dev.Board nodeMCU 1.0

NodeMCU 1.0 merupakan development board berbasis ESP8266 12-E. Board ini mempunyai satu buah channel analog ADC 10-bit (A0) dan 8 buah channel digital (D0-D7). Tegangan operasi board ini adalah 3.3 Volt dengan range tegangan input (pada pin VIN) 7-12 Volt. Board ini dapat dioperasikan dalam mode AP (Access Point); Station (Client); dan AP+STA.

Software

Framework Ionic

Framework Ionic adalah framework HTML5 yang membantu kita dalam mengembangkan aplikasi mobile dengan teknologi web seperti HTML, CSS dan Javascript. Framework Ionic menggunakan AngularJS sebagai framework berbasis web dan menggunakan Cordova untuk membangun aplikasi mobile. Framework Ionic platform memiliki lisensi opensource.

Cloud Database ThingSpeak.com

ThingSpeak merupakan open source "Internet of Things" aplikasi dan API untuk menyimpan dan mengambil data menggunakan HTTP melalui Internet atau melalui Local Area Network. Selain menyimpan dan mengambil data numerik dan alfanumerik, API ThingSpeak memungkinkan untuk pengolahan data numerik seperti timescaling, rata-rata, median, menjumlahkan, dan pembulatan. Setiap Saluran ThingSpeak mendukung entri data hingga 8 bidang data. Channel feed dukungan JSON, XML, dan format CSV untuk integrasi ke dalam aplikasi.
